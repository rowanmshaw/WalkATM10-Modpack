# WalkATM10-Modpack
A slightly modified version of ALL the Mods 10 for Walk's server.
Required to play on the server!
Server IP: walkatm10.playit.plus (Check out the Playit.gg service the people who made it are cool as hell)
-----------------------
Instructions to install and run:
1. - Download the pack file to your downloads folder.
2. - Download and install Curseforge if you haven't already.
3. - Hit the Import button towards the top of the Curseforge screen.
4. - Select the pack file from your Downloads folder.
5. - Launch the game and join the server with the IP above.
